### Getting started

1. Clone the repo
```sh
git clone https://github.com/pietrushka/example-tron-wallets
```
2. Install NPM packages
```sh
npm install
```
3. Start app
```sh
npm start
```
4. Open http://localhost:3000 to view it in the browser.
